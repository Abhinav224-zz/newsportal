# News Portal Frontend

A responsive React + Vite news portal frontend connected to a Node.js backend API.

## Features

- Public pages: Home (Hero + General section), Category page, Single news page
- Admin panel: Login, Create/Edit/Schedule/In-review/Publish/Delete posts
- Admin utilities: My Profile, Change Password, Sign Out
- Mobile-friendly clean UI

## Tech Stack

- React (Vite)
- Axios
- React Router

## Run Locally

```bash
npm install
npm run dev
```
